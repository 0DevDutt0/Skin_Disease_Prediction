# 1_Dremassist

A Skin Disease Prediction application.

## Prerequisites

- [Python 3.x](https://www.python.org/downloads/) installed.

## Installation

1.  Open a terminal in the project directory.
2.  Install the required dependencies:

    ```bash
    pip install -r requirements.txt
    ```

## Usage

1.  Run the application:

    ```bash
    python app.py
    ```

2.  Wait for the message `Running on http://127.0.0.1:5000` (or similar).
3.  Open your web browser and go to: [http://localhost:5000](http://localhost:5000)
4.  Upload an image to get a prediction and explanation.
